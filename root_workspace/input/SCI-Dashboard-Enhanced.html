Sprint Completion Index (SCI) Dashboard

Export your sprint backlog from Azure DevOps (Boards → Backlogs → Export to Excel) and connect it here. The dashboard automatically classifies tasks into SCI phases, calculates completion, and generates real-time metrics.

Expected columns in export:

ID · Title · Work Item Type · State · Tags · Assigned To · Remaining Work · Effort · Value Area · Iteration Path

Supports ADO backlog export (.xlsx) · Auto-classifies tasks into Design / Dev/UT / Test+ / Test− · Data stays local

⚠ Parent Column Missing

Your export does not include a Parent column. Tasks are being assigned to PBIs by sprint matching (approximate). For accurate SCI scores, add the Parent column to your Azure DevOps query: Column Options → search “Parent” → Add → Re-export
